KHALID'S DREAMS — IMAGES FOLDER
================================

এই ফোল্ডারে আপনার পণ্যের ছবি রাখুন।

ছবি যোগ করার ২টি উপায়:

উপায় ১ — Admin Panel থেকে (সহজ):
-----------------------------------
১. admin.html খুলুন
২. লগইন করুন (admin / khalid2025)
৩. পণ্য ম্যানেজ → পণ্যে ক্লিক করুন → ছবি আপলোড করুন
৪. ছবি Base64 ফরম্যাটে localStorage-এ সেভ হবে
৫. সাইটে সাথে সাথে দেখা যাবে

উপায় ২ — ম্যানুয়াল (products.js এডিট):
-----------------------------------------
১. ছবি এই ফোল্ডারে রাখুন (যেমন: honey.jpg)
২. products.js খুলুন
৩. সংশ্লিষ্ট পণ্যের image ফিল্ডে পাথ দিন:
   image: "images/honey.jpg"
