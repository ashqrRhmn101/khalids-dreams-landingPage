# 🍯 Khalid's Dreams — সম্পূর্ণ গাইড (v2)

## 📁 ফাইল স্ট্রাকচার

```
khalids-dreams/
├── index.html          ← মেইন ল্যান্ডিং পেজ
├── admin.html          ← অ্যাডমিন প্যানেল
├── css/
│   ├── style.css       ← সাইটের ডিজাইন
│   └── admin.css       ← অ্যাডমিনের ডিজাইন
├── js/
│   ├── products.js     ← সব ডেটা (পণ্য, ব্যানার, সেটিংস)
│   ├── app.js          ← সাইটের কার্যক্রম
│   └── admin.js        ← অ্যাডমিনের কার্যক্রম
├── images/
│   ├── README.txt      ← ছবি যোগের নির্দেশনা
│   └── (আপনার ছবি)
└── README.md           ← এই ফাইল
```

## 🚀 কিভাবে চালাবেন

সব ফাইল একই ফোল্ডারে রাখুন এবং `index.html` ব্রাউজারে খুলুন।

---

### অ্যাডমিন থেকে যা করতে পারবেন:
| ফিচার | বিবরণ |
|---|---|
| 📦 পণ্য যোগ/এডিট | ছবি, দাম, বিবরণ, ব্যাজ সহ |
| 🛒 অর্ডার ম্যানেজ | পেন্ডিং → সম্পন্ন/বাতিল |
| 🏷️ ক্যাটাগরি | নতুন ক্যাটাগরি ছবি সহ |
| 🎠 অফার ব্যানার | ছবি/ইমোজি সহ ব্যানার |
| 🖼️ হিরো স্লাইড | মেইন ব্যানার ম্যানেজ |
| 📈 Analytics | ভিজিট, অর্ডার, কার্ড ট্র্যাকিং |
| ⚙️ সেটিংস | WhatsApp, ডেলিভারি চার্জ, Pixel ID |

---

> **Note:** একই ব্রাউজারে খুললে সাইট লাইভ আপডেট হবে।
